# Auralis is a two-part project; Python dependencies live with the backend.
#
# This root file is intentionally a pointer, not a dependency list, to avoid a
# third (and drifting) source of truth. Install from the backend instead:
#
#   Full training / evaluation / export / ingestion environment:
#       pip install -r auralis-back/requirements.txt
#
#   Lean API / inference runtime only (what the Docker image uses):
#       pip install -r auralis-back/requirements.api.txt
#
# The frontend (auralis-front/) uses npm; see auralis-front/package.json.
